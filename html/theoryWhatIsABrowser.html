What is a Browser? How Does it Work?

27 June 2020

“A web browser takes you anywhere on the internet, letting you see text, images and video from anywhere in the world.” (Mozilla Developer)

Web browsers retrieve information from other parts of the web and displays is on whatever device you are using, it desktop or mobile. This information is transferred through the use of Hypertext Transfer Protocol. Hypertext Transfer Protocol (HTTP) is an application-layer protocol used in the transmitting of hypermedia documents.

This information then needs to be shared and displayed in a way that is formatted consistently so that anyone using the browser will be able to access that information. However, not all browser makers adopt this way of formatting resulting in the information being represented differently based on what web browser is being used. This then creates something known as web standards. Web standards which is the idea or the concept of creating a consistency between all browsers to benefit the users.

Browsers use Hypertext Markup Language (HTML) to create what is seen, heard and experienced on the web. HTML is the standard markup language used for documents designed to be displayed using a web browser. Web browsers will receive HTML documents from a web server or from local storage and will then render those documents into multimedia pages.
