Metadata - What? Why? How?

Metadata is, in its simplest definition, data that describes data. It is data providing information about one or more aspects of the data and to summarise basic information about the data which makes it easier to track and work with specific data.

Metadata can be stored and managed in a database called a metadata repository. Without context and a point of reference, it could be impossible to identify metadata just by looking at it. For example, a string of thirteen numbers could mean anything if context is not given, however if context was given, that the database is a log of books, the string of numbers can be identified as an ISBNs number that would be referring to a specific book within the database.
